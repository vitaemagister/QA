# QA Templates

Ready-to-use templates for QA documentation (English and Ukrainian versions).

## Contents

| Template | File | Format |
|----------|------|--------|
| Bug Report | `bug_report_template_en.docx` | Word |
| Bug Report (UA) | `bug_report_template_UA.docx` | Word |
| Test Case | `test_case_en.xlsx` | Excel |
| Test Plan | `test_plan_en.docx` | Word |
| Test Plan (UA) | `test_plan_ua.docx` | Word |

## Compatibility

- Microsoft Word / Excel
- Google Docs / Sheets
- LibreOffice Writer / Calc
